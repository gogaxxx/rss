Описание конфигурационных переменных
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

root 		- корень всей файловой иерархии проекта
config-dir 	- каталог, где находятся каталоги-конфигурации для отдельных
			фидов (*** поэтому надо поменять имя переменной используемой 
			для индивидуального каталога фида)
read-dir 	- каталог, где находятся готовые файлы для чтения
imgdir 		- каталог, где находятся картинки для, указывается относительно read-dir
			(*** разбить на два - img-dir где хранятся картинки и img-url -
			урл этого каталога для вставки в хтмл)
cache-root 	- каталог, где находятся файлы скачанные из интернеты (*** поменять
			на cache-dir, чтоб было одинаково)
