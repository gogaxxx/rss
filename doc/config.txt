Описание конфигурационных переменных
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

root 		- корень всей файловой иерархии проекта
config-dir 	- каталог, где находятся каталоги-конфигурации для отдельных
			фидов 
feed-dir	- каталог, где находится конфигурация конкретного фида, а также
			вспомогательные файлы guids, number и т.д.
read-dir 	- каталог, где находятся готовые файлы для чтения
imgdir 		- каталог, где находятся картинки, указывается относительно read-dir
			(*** разбить на два - img-dir где хранятся картинки и img-url -
			урл этого каталога для вставки в хтмл)
cache-dir 	- каталог, где находятся файлы скачанные из интернета

disabled    - этот фид отключён, не нужно его обрабатывать, любой скрипт
				должен сразу отбросить этот фид
